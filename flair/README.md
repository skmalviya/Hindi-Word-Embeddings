# How to use pretrained hindi flair-models

## Follow the flair_hindi.py program

* install flair
```
pip install flair
```

* import required libraries in your program
```
from flair.embeddings import WordEmbeddings, CharacterEmbeddings, FlairEmbeddings, StackedEmbeddings
from flair.data import Sentence
```

* Use the pre-trained Hindi WordEmbeddings models:
```
hindi_wikipedia_embeddings = WordEmbeddings('hi')
hindi_webcrawl_embeddings = WordEmbeddings('hi-crawl')
hindi_flair_forward = FlairEmbeddings('hi-forward')
hindi_flair_backword = FlairEmbeddings('hi-backward')
```

Note: First will download and load the FastText WE trained on hindi_wikipedia and webcrawl sources. Last will make available Flair WE trained on Wikipedia dumps + OPUS (https://github.com/flairNLP/flair/issues/614)

* Input the sentence and use the desired model to get WE.
```
sentence = Sentence('घास हरी है । ')
hindi_wikipedia_embeddings.embed(sentence)
```
